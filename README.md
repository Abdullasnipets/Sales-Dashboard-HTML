# Sales-Dashboard-HTML
A modern, fully interactive Sales Analytics Dashboard built using HTML, CSS, JavaScript, and Chart.js. This project visualizes business performance data from a Superstore dataset, helping users analyze sales, profit, orders, and regional trends in a clean and visually appealing interface.

🚀 Features
📈 KPI Overview
Total Sales
Total Profit
Total Orders
Profit Margin
📊 Interactive Charts
Monthly Sales & Profit Trend (Line Chart)
Sales Distribution by Category (Donut Chart)
Sub-category Performance (Bar Chart)
🌍 Region Analysis
Sales performance by region with dynamic progress bars
🏆 Top Products Table
Top 10 products ranked by profit
Includes sales, profit, and margin insights
🎛️ Dynamic Filters
Filter by Year (2021–2024)
Filter by Region
Filter by Category
⚡ Real-time UI Updates
Dashboard updates instantly based on selected filters
🛠️ Tech Stack
HTML5
CSS3 (Modern UI + Responsive Design)
JavaScript (Vanilla JS)
Chart.js (Data Visualization Library)
🎯 Purpose of the Project

This project is designed to:

Demonstrate data analytics and visualization skills

📂 Dataset
Superstore Sales Dataset (inspired by Kaggle dataset)
